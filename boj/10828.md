# 백준

## 10828번 스택

[문제링크](https://www.acmicpc.net/problem/10828)

스택을 구현하는 문제


### 코드

```c++
# include <iostream>
# include <vector>

using namespace std;

void push();
void pop();
void size();
void empty();
void top();

vector<int> stack;

int main() {
    int num; string command; int integer;

    cin >> num;

    for (int i = 0; i < num; i++) {
        cin >> command;

        if ( command == "push" ) { push(); }
        else if ( command == "pop" ) { pop(); } 
        else if (command == "size") { size(); } 
        else if (command == "empty") { empty(); } 
        else if (command == "top") { top(); }
    }
}

void push() {
    int num;
    cin >> num;
    stack.push_back(num);
}

void pop() {
    if (stack.size() > 0) {
        cout << stack.back() << endl;
        stack.pop_back();
    } else {
        cout << -1 << endl;
    }
}

void size() {
    cout << stack.size() << endl;
}

void empty() {
    cout << (stack.size() == 0) << endl;
}

void top() {
    if (stack.size() > 0) {
        cout << stack.back() << endl;
    } else {
        cout << -1 << endl;
    }
}
```

### 기타

`# include <stack>` 을 사용할 수도 있지만, `vector`을 익힐 겸 사용해봤다
