# 백준

## 10872번 : 팩토리얼

[문제링크](https://www.acmicpc.net/problem/10872)

 N!을 출력하는 프로그램


### 코드

```c++
#include <iostream>

using namespace std;

int factorial(int n);

int main() {
    ios::sync_with_stdio(false); cin.tie(NULL); cout.tie(NULL);
    int n;
    cin >> n;
    cout << factorial(n);
}

int factorial(int n) {
    if (n == 0) { return 1; }
    else if (n > 0) { return factorial(n - 1) * n; }
}
```
